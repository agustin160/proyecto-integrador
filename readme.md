# Mi Rutina de Gimnasio
Esta conformado por tablas el la cual se encuentra mi rutina de gimnasio

## Descripción del Proyecto

La aplicación consiste en una guía de entrenamiento enfocada en musculación, donde se muestra una rutina semanal organizada por días, grupos musculares, ejercicios, series, pesos y frecuencia de entrenamiento.

Además, incluye recomendaciones para crear una rutina personalizada, consejos sobre entrenamiento, información sobre alimentación y un formulario de contacto para solicitar una rutina.

### Diseño Visual

Paleta de colores en tonos azules y verdes.
Uso de variables CSS para colores y fuentes.
Secciones con bordes redondeados y sombras.
Menú de navegación con Flexbox.
Efectos hover y transiciones en enlaces y botones.
Tablas estilizadas con colores alternados.
Formulario visualmente mejorado.
Uso de Grid para organizar el contenido principal.
